# Eklips Archive

This folder contains the latest build of each Eklips/Sol version.
Eklips had only became accessible in Github at v3.0.

v1.0 is really old and lost to time, it was used to attempt making a Progressbar95 clone but got scrapped in favor of v2.0.

Then v3.0 was a major modification to v2.0 to the point of not being API compatible, then so was v3.1 but still kept *SOME* backwards-compatibility.

Then 4.0A came in, a complete rewrite of Eklips which got superseeded by 5.0A in a couple of months, which was.. another rewrite.

Before 5.0A, i got locked out of my old account, and now Eklips is in this one. Whoops.
